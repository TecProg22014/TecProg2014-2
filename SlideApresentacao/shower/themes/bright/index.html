O que é?

O que é o GitLab

  • Ferramenta de gerenciamento de repositórios git em um servidor centralizado.
  • Que nos permite:
  • Controle completo de seus repositórios
  • Controle da visibilidade dos projetos (Public / Private)

Funcionalidades

Principais Funcionalidades

  • Issue Tracking
  • Wiki
  • Graphs
  • Protected Branches
  • Web Hooks
  • Snippets
